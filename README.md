# Endless Runner (3D) – Unity


A 3D endless runner prototype developed in Unity. The project includes a lane-based spawning system, obstacles, coin pickups, object pooling for performance, UI flow, and camera follow.

### Features
- Endless runner track logic
- Lane pattern spawner for obstacles and pickups
- Coin collection system
- Object pooling (SimplePool / IPoolable)
- Player controller + camera follow
- Basic UI manager

### Tech Stack
- Unity **6000.3.4f1**
- C#

### How to Run
1. Open the project in Unity **6000.3.4f1**
2. Open the scene: `Assets/Scenes/SampleScene.unity`
3. Press Play

### Project Structure
- `Assets/_Project/Scripts/` – Core gameplay scripts (spawning, track, player, UI)
- `Packages/` – Unity packages
- `ProjectSettings/` – Unity project configuration
